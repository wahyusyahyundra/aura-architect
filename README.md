# 🌌 Aura Architect - AI Life Planner

**Aura Architect** is a premium, AI-powered life planning application that helps users crystallize their vague ambitions into structured, actionable blueprints. Featuring **Aura**, a celestial-themed AI agent, the application bridges the gap between dreaming and doing.

![Aura Preview](public/assets/aura.png)

## Key Features
- **Meaningful AI Integration**: Unlike standard chatbots, Aura generates structured JSON data that is rendered into an interactive roadmap.
- **Goal Crystallization**: Converts natural language ambitions into milestones, tasks, and durations.
- **Premium Aesthetics**: Built with a high-end glassmorphism design, custom animations with Framer Motion, and a unique character identity.
- **Character-Driven UX**: Features Aura, a wise celestial agent who guides users through their planning process.

## Tech Stack
- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **AI Engine**: OpenRouter (Gemini 2.0 Flash)
- **State Management**: React Hooks

## Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/wahyusyahyundra/aura-architect
   cd web-ai
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env` file in the root directory and add your OpenRouter API Key:
   ```env
   REACT_APP_OPENROUTER_API_KEY=your_api_key_here
   ```

4. **Run the application**:
   ```bash
   npm start
   ```
